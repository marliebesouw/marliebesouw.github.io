A new use case is initiated following the Project AIR framework: fracture detection on X-rays. The status of this project is still on preparing, since we are now working on building a multi-center datacase which we can use for the evaluation.
Vendors with a fracture detection algorithm will be contacted in a later stadium for participation. 
The protocol can already be found <a href="https://marliebesouw.github.io/protocols/">here</a>. If you have any questions or uncertainties, please don't hesitate to <a href="mailto:marlie.besouw@radboudumc.nl">contact</a> us.
